# ADNz Frontend Scaffold

Minimal Next.js 15 + TypeScript + Tailwind starter that mirrors the requested frontend folder structure.

## Getting started

1. Install dependencies once internet access is restored:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```

## Notes
- Global styles live in `src/app/globals.css` and use Tailwind directives.
- The layout ensures a responsive container and sets Vietnamese locale metadata.
